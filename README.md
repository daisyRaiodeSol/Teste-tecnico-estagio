Parte 4
1. O HTML é uma linguagem de marcação que serve para criar a estrutura de uma página web.
  É ela que diz ao navegador o que a página irá fazer.

2. A diferença entre id e class é que id serve de identificador único para cada elemento html específico
   ao passo que a class serve para classificar um grupo de elementos.

3. O javascript é uma linguagem de programação usada para criar dinamismo às oáginas web. O Javascript
   roda no navegador(browser).

4. O PHP é uma linguagem usada para processar dados no servidor. Cria páginas web em tempo real e as envia
   ao visitante do site.

5. O $_POST é uma variável especial do PHP usada para colectar dados enviados por um formulário usando o
   método POST. Ele guarda os valores que o utilizador preencheu no formulário.

6. O Git serve para controlar versões do cógido. Ele guarda o histórico de um projecto e permite acompanhar
   todas as alterações feitas ao logo do tempo.


   Explicação do que foi feito neste projecto:

  ##Tecnologias Utilizadas
   - HTML
   - CSS
   - JavaScript
   - PHP
   - XAMMP (Apache)

    ##Estrutura do Projecto
   Cadastro/
   |___index.html
   |___processa.php
   |___script.js
   |___style.css

    ##Funcionalidades
    - Formulário com os campos:
      - Nome
      - E-mail
      - Senha
    -Validação com Javascript:
      - Verifica se os campos estão preenchidos
      - Mostra um alerta se algum campo estiver vazio
    -Processamento em PHP:
      - Recebe os dados via POST
      - Verifica se os campos estão vazios
      - Exibe os dados enviados

      ##Como Executar
      1. Instalar o XAAMP
      2. Colocar a pasta do projecto em: C:\xampp\htdocs\
      3. Iniciar o Apache
      4. Aceder pelo navegador: http://localhost/Cadastro/index.html
